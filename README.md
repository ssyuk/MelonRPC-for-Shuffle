# 정보
Spotify는 RPC를 지원하지만, 멜론에서는 RPC를 지원하지 않아요..<br>
하지만 이 프로젝트를 사용하면, 멜론에서도 RPC를 사용할 수 있어요!
<br>

# 설치
```
1. appmgr repo add https://raw.githubusercontent.com/ssyuk/shflrepo/main/repo.json
2. appmgr add melonrpc
```
